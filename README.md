# VoltFuel

App web simples — HTML, CSS e JS puro, sem framework, sem build tool.

## Estrutura

```
├── index.html       # boilerplate mínimo com link para styles.css e app.js
├── styles.css       # reset + body dark base
├── app.js           # 'use strict', monta no #app
├── README.md        # documentação da estrutura e como rodar
├── .gitignore       # .DS_Store, Thumbs.db
└── assets/          # pasta pronta para imagens/favicon/fontes
```

## Como rodar

Abra `index.html` direto no browser.
