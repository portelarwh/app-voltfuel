# VoltFuel

App web simples — HTML, CSS e JS puro, sem framework, sem build tool.

## Estrutura

```
├── index.html      # estrutura
├── styles.css      # estilo
├── app.js          # lógica
├── assets/         # imagens, favicon, fontes
└── .gitignore
```

## Como rodar

Abra `index.html` direto no browser.
